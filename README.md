# A24 Press Site

Bootstrapped with React-Create-App. Using Netlify Identity for the Auth.

When a user registered they are added to a mailchimp listed specifically associated to this process.

## Local Dev Setup

- Clone repo locally and `npm install`
- `npm run start` to start the dev server and watch for changes
- Development of the back-end / Sanity Studio is managed through the [press-api](https://github.com/a24films/press-api) repo.
